O trabalho consistirá em um site que faz uso de 3 tecnologias: HTML, CSS e Javascript.
Todas as tecnolofias devem estar nas suas versões mais recentes;
O tema pode ser qualquer um, inclusive um já usado na disciplina;


O site deve ser publicado em algum servidor externo de preferência da equipe (githubpages, por exemplo)
O HTML deve ser semântico
O CSS deve incluir animações, transições e transformações;
Deve ter menu dropdown e layout responsivo
Ter pelo menos 1 formulário
O DOM deve ser manipulado de alguma forma usando JS
Deve usar o AXIOS para a leitura de dados
O site todo deve ter no mínimo 5 páginas diferentes
